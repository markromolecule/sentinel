# Agent Rules Configuration

The rules for this project are located in:
`context-factory/web/rules/`

One file per rule category (e.g., `data-access-layer.md`, `project-structure.md`, `components.md`, `mutation-hooks.md`,
`query-hooks.md`, `service-layer.md`, `zustand-store.md`, `api-routes.md`).

## Applying Rules

The agent should natively check the `context-factory/web/rules/` directory for active rules.
